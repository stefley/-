## 数组定长

----------

数组是定长的，js引擎会自动帮我们扩容，内存是操作系统来分配的

尽量要设置好数组长度，比如分页和贪吃蛇

## 数组特性

-----------

1.存储在物理空间上是连续的

2.底层数组长度不可变

3.数组变量执向第一个元素的位置（数组下标）

> 通过偏移查询数据性能是最好的

## 优点

----

查询性能好

## 缺点

----

1. 空间必须是连续的，数组比较大，当系统的碎片空间较多的时候，容易存不下
2. 因为数组长度是固定的，所以数组的内容难以被添加和删除，添加一个东西和移除某一个性能都是极度消耗的

## js中创建数组

```js
let a = [1,2,3]
let b = new Array(10) // 确定数组长度  10个元素
```
